Supports

- Webpack
- Material UI
- Hot Module replacement
- React Hotloading (persists state)
- Recommended folder structure for a react.js webapp project 

### How to start running
Run these commands

To install the required dependencies 

	$ npm install
	
To run the app
 	
	$ npm run serv
